# Markdown Guide

## Headings

```markdown
# Big Heading
## Smaller heading
### Even smaller heading
#### Even smaller heading
```

## Bold

```markdown
**Bold text** not bold text
```

## Italic

```markdown
*Italic text* not italic text
```

## Underline

```markdown
__underlined__ not underlined
```

## Links

```markdown
You can just paste the URL: https://google.com
Or, you can make it [with custom text](https://google.com)
```
